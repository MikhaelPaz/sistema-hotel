# 🏨 Sistema de Hotel

Sistema de gerenciamento de hotel desenvolvido com:

- Node.js
- Express
- MySQL
- HTML
- CSS
- JavaScript

## 🚀 Funcionalidades

- Cadastro de hóspedes
- Check-out com cálculo automático
- Dashboard com faturamento mensal
- Limpar faturamento mensal
- Controle de reservas ativas
